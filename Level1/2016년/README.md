## %연산자, switch문 사용
```
switch 문을 활용해서 날짜를 출력한다
switch(변수){
    case 변수1:
        break;
    case 변수2:
        break;
    case 변수3:
        break;
    default:
}

%는 모듈러 연산이라고 하며 나머지를 반환한다
```
