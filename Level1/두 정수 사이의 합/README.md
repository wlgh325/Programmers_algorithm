## Math.max와 Math.min 함수 이용
```
Math는 java.lang.Math를 import 하면 사용 가능!!
아니면 Math.max() 이런식으로 직접 사용할 수 있다

1. java 최댓값(max) 구하기
max = Math.max(a,b)

2. java 최솟값(min) 구하기
min = Math.min(a,b)
```
